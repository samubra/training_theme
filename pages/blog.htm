The header have important colors, texts and icons that will define your app's identity, so it's a key element on your website.

Header bar

To start, just add an element with "mc-header" class into your app's "body". This will set the space for your main panel.

You can add several lines to your header by adding a "mc-header-line" element class to its children.

  • <header class="mc-header">
    • <div class="mc-header-line"></div>
    • <div class="mc-header-line"></div>
  • </header>

Header content

You can place Icon Buttons and Titles into this bar. Also you can place a div with the "mc-right" class to place the icons on the right side.

  • <header class="mc-header">
    • <div class="mc-header-line">
      • <span class="mc-button-icon" mc-action="menu"></span>
      • <p class="mc-title"> Bars </p>
      • <div class="mc-right">
        • <span class="mc-button-icon" mc-action="download"></span>
        • <span class="mc-button-icon" mc-action="share"></span>
      • </div>
    • </div>
  • </header>

Tabs

Tabs are designed to navigate between sections on the same catergory. To add a tabs bar, just place an element with the class "mc-tabs-bar" like direct children of your "mc-header".

Then, place as "mc-tab" as you want into it.

  • <header class="mc-header">
    • <div class="mc-header-line">
      • ...
    • </div>
    • <div class="mc-tabs-bar">
      • <a href="#" class="mc-tab">Tab name</a>
      • <a href="#" class="mc-tab">Tab name</a>
    • </div>
  • </header>

If you add the "mc-selected" class, the tab will display an indicator line on its bottom.
